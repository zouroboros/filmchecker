About FilmChecker

FilmChecker is free and open source. It comes under the MIT-Licence. You can find the source code at Github.

If you have any suggestions, feedback or ideas for improvments feel free to open an issue in our issue tracker or write an email to filmchecker@murks.me.

FilmChecker uses the following libraries:

  1. Guava: Google Core Libraries for Java
  2. jsoup: Java HTML Parser
  3. AndroidX and SwipeRefreshLayout and ConstrainedLayout packages.
  4. Material Components for Android
